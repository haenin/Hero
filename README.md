HERO는 기업 내에 분산된 **근태 · 휴가 · 평가 · 결재 · 급여 데이터**를 하나의 흐름으로 통합 관리하는 **HR 시스템**입니다.  
**Role 기반 권한 관리**로 사용자(관리자/부서장/사원)별로 필요한 기능만 노출되도록 설계했습니다.
